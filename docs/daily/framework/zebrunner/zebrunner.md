zebrunner是一个Github的自动化测试的配置解决方案，里面包含了从很多有用的组件:

1. Jenkins Master/Slaves Nodes配置
2. Selenium Hub配置
3. STF: Mobile Device Farm 手机管理云工具
4. SonarQube配置
5. 报告Reporting

Zebrunner把所有自动化测试相关的组件整合起来变成自动化测试的基础设施。
